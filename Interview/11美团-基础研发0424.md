## 统计页面中dom节点总数function count (element) {}返回值 number
```
function count(element){
    let arr=element.children;
    let number=0;
    if(arr.length==0){
        return 0
    }
    for(let i=0;i<arr.length;i++){
        number=number+count(element.children[i]);
    }
    return number;
}
```
