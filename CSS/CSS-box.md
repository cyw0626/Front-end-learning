# CSS框模型
**CSS框模型规定了元素框处理元素内容、内边距、边框和外边距的方式。**  
元素框的最内部分是实际的内容，直接包围内容的是内边距。  
内边距呈现了元素的背景，内边距的边缘是边框。  
边框以外是外边距，外边距默认透明。  
- padding：内边距
- margin：外边距
## CSS内边距
CSS padding属性定义元素边框与元素内容之间的空白区域。
- padding-top //上内边距
- padding-right //右内边距
- padding-bottom  //下内边距
- padding-left  //左内边距
>属性值为百分数值时，相对于父元素来计算。
